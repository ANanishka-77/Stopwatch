# Stopwatch

⏱️ Stopwatch Web Application
📌 Project Description

This is a simple and interactive Stopwatch web application built using HTML, CSS, and JavaScript. It allows users to measure time accurately with features like start, pause, reset, and lap recording.

🚀 Features
▶️ Start and Pause functionality
🔄 Reset the stopwatch
🏁 Record lap times
⏱️ Displays time in hours : minutes : seconds : milliseconds
📱 Responsive and user-friendly interface



🛠️ Technologies Used
HTML (Structure)
CSS (Styling)
JavaScript (Functionality & Logic)
